# Group3 About Us
### 1. 安裝套件
    $ yarn
    
### 2. pug 轉檔為 html
    $ yarn pug3 index.pug -o ./dist/ -P --watch
    
### 3. sass 轉檔為 css 
    $ yarn node-sass index.pug -o ./dist/ -P --watch
    
### 檔案位置  
```
1. pug、sass 在主頁面
2. js 和編譯後的html、css 在 dist 資料夾底下
```
